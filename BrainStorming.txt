----------------------------------------------------------------
GLOBAL RULES
----------------------------------------------------------------

- Dark and light mode on every single page
- English (LTR) and Arabic (RTL) toggle
  * Affects: all UI text, headers, nav, buttons, labels
  * Does NOT affect: blog post content, book content, comments
- Fully responsive on all screen sizes (mobile, tablet, desktop)
- Header is fixed across the entire website
- Authentication: Google only
  * No sign in / sign up pages
  * Just a Sign In button that redirects to Google


----------------------------------------------------------------
HEADER — appears on all pages
----------------------------------------------------------------

Left side:
  - Logo: MMM (styled)

Middle:
  - Nav links: Services | Books | Blogs | Social Media | Dashboard ( If Admin )

Right side:
  - Language toggle: EN / AR selector
  - Dark / Light mode toggle switch
  - If logged out:
      * Sign In button (redirects to Google)
  - If logged in as regular user:
      * Profile picture replaces Sign In button
      * Click picture → dropdown appears with:
          - Profile image 
          - Name
          - Email
          - Edit button → clicking it turns name into
            a text input with and a small text under profile image say upload image ( user can upload image or edit the name or both and then save ) Save button
  - If logged in as admin (me):
      * Same profile dropdown as above
      * Email icon appears in header
      * If there are unread emails → red circle on icon
      * Click email icon → goes to Email Page
