# 🚀 Mahmoud Elrashedy | Full Stack & AI Automation Portfolio

A high-performance, bilingual (English & Arabic) personal website built with **Next.js 15**, **TypeScript**, and **Tailwind CSS**. This project is engineered for maximum SEO visibility to rank for "Mahmoud Elrashedy" and "محمود الرشيدي" globally and in the Middle East.

---

## 🌟 Key Features

* **🌐 100% Bilingual SEO:** Advanced metadata configuration using `generateMetadata` for both English and Arabic search intent.
* **📚 Book Summaries Engine:** A dedicated section for summarized literature, optimized for sitelink indexing on Google.
* **✍️ Dynamic Blog:** Integration for technical and personal writings with social media preview cards (Open Graph).
* **🤖 AI & Automation Ready:** Built to showcase vision models, automation scripts, and full-stack software engineering.
* **🎨 UI/UX Focused:** Designed with Figma-level precision, supporting Right-To-Left (RTL) for Arabic and LTR for English.

---

## 🛠️ Tech Stack

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Images:** [Cloudinary](https://cloudinary.com/)
* **Deployment:** [Vercel](https://vercel.com/)
